Meta tags, CSS, JavaScript e todas as configurações essenciais

🎯 O que você aprenderá: Como configurar corretamente o <head> do HTML com meta tags, links CSS, scripts JavaScript e otimizações para SEO e performance.

1. Estrutura Básica do Head

1.1 Template Completo

Aqui está um template completo de <head> que você pode usar como base:

<!DOCTYPE html>
<html lang="pt-BR">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <meta name="description" content="Descrição da sua página">
    <meta name="keywords" content="palavra-chave1, palavra-chave2, palavra-chave3">
    <meta name="author" content="Seu Nome">
    
    <title>Título da Página</title>
    
    <!-- Favicon -->
    <link rel="icon" type="image/x-icon" href="favicon.ico">
    
    <!-- CSS -->
    <link rel="stylesheet" href="styles.css">
    
    <!-- JavaScript -->
    <script src="script.js" defer></script>
</head>

2. Meta Tags Essenciais

M Charset e Viewport

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Obrigatórias para funcionamento correto em dispositivos móveis.

M SEO Básico

<meta name="description" content="Sua descrição aqui (máx 160 caracteres)">
<meta name="keywords" content="html, css, javascript, tutorial">
<meta name="author" content="Seu Nome">

Fundamentais para SEO e busca no Google.

M Open Graph (Redes Sociais)

<meta property="og:title" content="Título para compartilhamento">
<meta property="og:description" content="Descrição para redes sociais">
<meta property="og:image" content="https://seusite.com/imagem.jpg">
<meta property="og:url" content="https://seusite.com">

Para Facebook, LinkedIn e outras redes sociais.

M Twitter Cards

<meta name="twitter:card" content="summary_large_image">
<meta name="twitter:title" content="Título para Twitter">
<meta name="twitter:description" content="Descrição para Twitter">
<meta name="twitter:image" content="https://seusite.com/imagem.jpg">

Otimização específica para Twitter.

3. Configurando CSS

3.1 Um Arquivo CSS

Para incluir um único arquivo CSS:

<link rel="stylesheet" href="styles.css">

Arquivo: styles.css

3.2 Múltiplos Arquivos CSS

Para incluir vários arquivos CSS (ordem importa!):

<!-- CSS base (primeiro) -->
<link rel="stylesheet" href="css/reset.css">
<link rel="stylesheet" href="css/base.css">

<!-- CSS específicos -->
<link rel="stylesheet" href="css/header.css">
<link rel="stylesheet" href="css/main.css">
<link rel="stylesheet" href="css/footer.css">

<!-- CSS responsivo (último) -->
<link rel="stylesheet" href="css/responsive.css">
💡 Dica: Organize em pastas: css/reset.css css/base.css css/components.css
3.3 CSS Externos (CDN)

Para incluir bibliotecas CSS de CDN:

<!-- Bootstrap -->
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css" rel="stylesheet">

<!-- Font Awesome -->
<link rel="stylesheet" href="https://cdnjs.cloudflare.com/ajax/libs/font-awesome/6.4.0/css/all.min.css">

<!-- Google Fonts -->
<link href="https://fonts.googleapis.com/css2?family=Inter:wght@300;400;600;700&display=swap" rel="stylesheet">

<!-- Seu CSS (sempre por último) -->
<link rel="stylesheet" href="styles.css">

4. Configurando JavaScript

4.1 Um Arquivo JavaScript

Para incluir um único arquivo JS:

<script src="script.js" defer></script>

Arquivo: script.js

✅ Atributo defer: Garante que o script execute após o HTML estar carregado.
4.2 Múltiplos Arquivos JavaScript

Para incluir vários arquivos JS:

<!-- Bibliotecas externas primeiro -->
<script src="js/vendor/jquery.min.js"></script>
<script src="js/vendor/bootstrap.bundle.min.js"></script>

<!-- Seus módulos JS -->
<script src="js/utils.js" defer></script>
<script src="js/components.js" defer></script>
<script src="js/main.js" defer></script>
💡 Estrutura sugerida: js/vendor/ (bibliotecas) js/modules/ (seus módulos) js/main.js (arquivo principal)
4.3 JavaScript com CDN

Para incluir bibliotecas JS de CDN:

<!-- jQuery -->
<script src="https://code.jquery.com/jquery-3.7.1.min.js"></script>

<!-- Bootstrap -->
<script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>

<!-- Chart.js -->
<script src="https://cdn.jsdelivr.net/npm/chart.js"></script>

<!-- Seus scripts (sempre por último) -->
<script src="js/app.js" defer></script>
4.4 Atributos Importantes
⚡ defer
<script src="script.js" defer></script>

Executa após o HTML carregar completamente.

🚀 async
<script src="analytics.js" async></script>

Carrega e executa em paralelo (para scripts independentes).

6. Otimizações e Melhores Práticas

6.1 Performance
  • Use defer para scripts que dependem do DOM
  • Use async para scripts independentes (analytics)
  • Coloque CSS no <head> e JS antes do </body>
  • Minimize o número de requests HTTP
  • Use CDN para bibliotecas populares
6.2 SEO
  • Sempre inclua meta description (máx 160 caracteres)
  • Use title descritivo e único por página
  • Configure Open Graph para redes sociais
  • Adicione meta tags de autor e keywords
  • Use favicon para melhorar a confiabilidade
