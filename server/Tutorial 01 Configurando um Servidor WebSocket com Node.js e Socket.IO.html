Tutorial: Configurando um Servidor WebSocket com Node.js e Socket.IO

Requisitos:
  • Node.js instalado no computador
  • Visual Studio Code (ou outro editor de código)
  • Conhecimentos básicos de JavaScript

1. Criando a Estrutura do Projeto

Primeiro, crie uma nova pasta chamada serverz para o projeto e abra-a no VS Code. Em seguida, abra o terminal e execute os seguintes comandos:

npm init -y

Este comando criará um arquivo package.json com as configurações padrão:

{
  "name": "serverz",
  "version": "1.0.0",
  "description": "",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC"
}

2. Instalando as Dependências

Agora vamos instalar o Socket.IO e Express:

npm install socket.io express

3. Criando o Servidor

Crie um arquivo chamado server.js na raiz do projeto com o seguinte código:

const express = require('express');
const app = express();
const http = require('http').createServer(app);
const io = require('socket.io')(http);

// Servir arquivos estáticos da pasta 'public'
app.use(express.static('public'));

// Configuração básica do Socket.IO
io.on('connection', (socket) => {
    console.log('Um usuário conectou');

    // Quando um usuário desconecta
    socket.on('disconnect', () => {
        console.log('Um usuário desconectou');
    });
});

// Iniciando o servidor na porta 3000
const PORT = process.env.PORT || 3000;
http.listen(PORT, () => {
    console.log(`Servidor rodando na porta ${PORT}`);
});

<body>
    <div class="container">
        <h1>Teste WebSocket</h1>
        <div id="status" class="disconnected">Desconectado</div>

        <div class="message-container">
            <input type="text" id="message" placeholder="Digite sua mensagem" />
            <button onclick="sendMessage()">Enviar</button>
        </div>

        <div id="messages"></div>
    </div>

    <script src="/socket.io/socket.io.js"></script>
    <script>
        const socket = io();
        const statusDiv = document.getElementById('status');
        const messagesDiv = document.getElementById('messages');
        const messageInput = document.getElementById('message');

        // Conexão estabelecida
        socket.on('connect', () => {
            statusDiv.textContent = 'Conectado';
            statusDiv.className = 'connected';
            addMessage('Conectado ao servidor', 'system');
        });

        // Desconexão
        socket.on('disconnect', () => {
            statusDiv.textContent = 'Desconectado';
            statusDiv.className = 'disconnected';
            addMessage('Desconectado do servidor', 'system');
        });

        // Receber mensagem
        socket.on('message', (msg) => {
            addMessage(`Servidor: ${msg}`);
        });

        // Enviar mensagem
        function sendMessage() {
            const message = messageInput.value.trim();
            if (message) {
                socket.emit('message', message);
                addMessage(`Você: ${message}`);
                messageInput.value = '';
            }
        }

        // Adicionar mensagem ao chat
        function addMessage(text, type = '') {
            const messageElement = document.createElement('div');
            messageElement.className = `message ${type}`;
            messageElement.textContent = text;
            messagesDiv.appendChild(messageElement);
            messagesDiv.scrollTop = messagesDiv.scrollHeight;
        }

        // Enter para enviar mensagem
        messageInput.addEventListener('keypress', (e) => {
            if (e.key === 'Enter') {
                sendMessage();
            }
        });
    </script>
</body>

5. Configurando o Script de Inicialização

Atualize o arquivo package.json para adicionar o script de início do servidor. Modifique a seção "scripts" para:

"scripts": {
    "start": "node server.js",
    "test": "echo \"Error: no test specified\" && exit 1"
},

6. Executando o Servidor

Para iniciar o servidor, execute:

npm start

Seu servidor estará rodando em http://localhost:3000 Abrir

Como Testar com Múltiplos Clientes

Para testar o servidor com múltiplos clientes simulando diferentes usuários:

  1. Abra uma janela normal do navegador e acesse http://localhost:3000
  2. Pressione Ctrl + Shift + N (Chrome/Edge) ou Ctrl + Shift + P (Firefox) para abrir uma janela anônima
  3. Na janela anônima, acesse também http://localhost:3000
  4. Repita o processo para simular mais clientes

Cada janela anônima será tratada como um cliente diferente pelo servidor WebSocket, permitindo testar a comunicação entre múltiplos usuários.
