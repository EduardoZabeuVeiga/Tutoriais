Tutorial: Atualizar Aplicativo na Microsoft Store

Procedimento para atualizar a versão do pacote no pubspec.yaml e enviar a compilação corrigida para a Microsoft Store.

Passo 1: Atualizar a versão no pubspec.yaml

Abra o arquivo pubspec.yaml do seu projeto Flutter.

Localize a seção msix_config e altere o valor de msix_version para uma numeração superior à anterior (exemplo: alterar de 1.0.0.0 para 1.1.0.0).

Passo 2: Limpar e compilar o projeto

Para garantir que não ocorram problemas de cache, execute os comandos de limpeza e compilação em sequência no seu terminal:

flutter clean
flutter build windows --release

Passo 3: Criar o novo pacote MSIX

Gere o arquivo empacotado (.msix) atualizado executando o comando abaixo:

flutter pub run msix:create

Passo 4: Enviar para o Microsoft Partner Center

Acesse o painel do seu aplicativo clicando no link abaixo:

Acessar Microsoft Partner Center (Link Clicável)

  1. Na tela de gerenciamento de Packages (Pacotes), localize o pacote antigo que apresentou erro.
  2. Clique no botão Delete localizado logo abaixo do nome do arquivo com erro.
  3. Arraste e solte o novo pacote .msix (gerado no Passo 3) para a área de envio.
