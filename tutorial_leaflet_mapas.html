Tutorial Leaflet: Criando Mapas Interativos

Aprenda a criar mapas interativos com marcadores, popups e controles

Introdução ao Leaflet

O Leaflet é uma das bibliotecas JavaScript mais populares para criação de mapas interativos. É conhecida por ser leve, ter uma API simples e funcionar bem em dispositivos móveis. Neste tutorial, você aprenderá:

  • Como criar e configurar um mapa básico
  • Adicionar marcadores, popups e formas geométricas
  • Personalizar ícones e estilos
  • Trabalhar com camadas e controles
  • Responder a eventos do mapa
  • Integrar diferentes provedores de mapas

1. Configuração Inicial

Para começar a usar o Leaflet, você precisa incluir os arquivos CSS e JavaScript necessários e criar um container para o mapa:

<link rel="stylesheet" href="https://unpkg.com/leaflet@1.9.4/dist/leaflet.css"
integrity="sha256-p4NxAoJBhIIN+hmNHrzRCf9tD/miZyoHS5obTRR9BMY="
crossorigin=""/>
<div id="map" style="height: 400px"></div>
<script src="https://unpkg.com/leaflet@1.9.4/dist/leaflet.js"
integrity="sha256-20nQCchB9co0qIjJZRGuk2/Z9VM+kNiyxNV1lvTlZBo="
crossorigin=""></script>
Importante: O container do mapa deve ter uma altura definida, caso contrário o mapa não será exibido corretamente.

2. Criando seu Primeiro Mapa

Vamos criar um mapa básico centrado em São Paulo:

const map = L.map('map').setView([-23.5505, -46.6333], 13);
L.tileLayer('https://tile.openstreetmap.org/{z}/{x}/{y}.png', {
    maxZoom: 19,
    attribution: '© <a href="https://www.openstreetmap.org/copyright">OpenStreetMap</a> contributors'
}).addTo(map);

Resultado:

Explicação:
  • L.map('map') - Cria uma instância do mapa no elemento com ID 'map'
  • setView([-23.5505, -46.6333], 13) - Define o centro do mapa (São Paulo) e o nível de zoom
  • L.tileLayer() - Adiciona uma camada de tiles do OpenStreetMap

3. Adicionando Marcadores e Popups

Marcadores são úteis para destacar locais específicos no mapa. Vamos adicionar um marcador com um popup informativo:

const marker = L.marker([-23.5505, -46.6333]).addTo(map);
marker.bindPopup("<b>São Paulo</b><br>A maior cidade do Brasil").openPopup();

Personalizando Marcadores

Você pode personalizar o ícone do marcador:

const iconePersonalizado = L.icon({
    iconUrl: 'caminho/para/seu/icone.png',
    iconSize: [38, 38],     // tamanho do ícone
    iconAnchor: [19, 38],   // ponto do ícone que corresponderá à localização
    popupAnchor: [0, -38]   // ponto de onde o popup deve abrir
});
L.marker([-23.5505, -46.6333], {icon: iconePersonalizado}).addTo(map);

4. Formas Geométricas

O Leaflet permite adicionar várias formas geométricas ao mapa:

const circulo = L.circle([-23.5505, -46.6333], {
    color: 'red',
    fillColor: '#f03',
    fillOpacity: 0.5,
    radius: 500
}).addTo(map);
const poligono = L.polygon([
    [-23.54, -46.63],
    [-23.55, -46.64],
    [-23.53, -46.65]
], {
    color: 'purple',
    fillColor: '#909',
    fillOpacity: 0.5
}).addTo(map);
const linha = L.polyline([
    [-23.54, -46.63],
    [-23.55, -46.64],
    [-23.53, -46.65]
], {
    color: 'blue',
    weight: 3
}).addTo(map);
Dica: Todas as formas podem ter popups e eventos associados a elas.

5. Eventos do Mapa

O Leaflet oferece vários eventos que você pode usar para tornar seu mapa interativo:

map.on('click', function(e) {
    const lat = e.latlng.lat.toFixed(4);
    const lng = e.latlng.lng.toFixed(4);
    L.popup()
        .setLatLng(e.latlng)
        .setContent(`Você clicou em: ${lat}, ${lng}`)
        .openOn(map);
});
map.on('zoomend', function() {
    const zoom = map.getZoom();
    console.log(`Nível de zoom atual: ${zoom}`);
});
map.on('moveend', function() {
    const centro = map.getCenter();
    console.log(`Centro do mapa: ${centro.lat}, ${centro.lng}`);
});

6. Camadas e Controles

Você pode adicionar diferentes camadas de mapas e permitir que o usuário alterne entre elas:

const osm = L.tileLayer('https://tile.openstreetmap.org/{z}/{x}/{y}.png', {
    maxZoom: 19,
    attribution: '© OpenStreetMap'
});
const satellite = L.tileLayer('https://server.arcgisonline.com/ArcGIS/rest/services/World_Imagery/MapServer/tile/{z}/{y}/{x}', {
    attribution: 'Tiles © Esri — Source: Esri, i-cubed, USDA, USGS, AEX, GeoEye, Getmapping, Aerogrid, IGN, IGP, UPR-EGP, and the GIS User Community'
});
const baseMaps = {
    "OpenStreetMap": osm,
    "Satélite": satellite
};

// Adiciona o controle de camadas
L.control.layers(baseMaps).addTo(map);

// Coordenadas do ponto de referência (São Paulo)
const pontoRef = {
    lat: -23.5505,
    lng: -46.6333
};

// Inicializa o mapa
const map = L.map('map').setView([pontoRef.lat, pontoRef.lng], 13);

// Adiciona camada de tiles
L.tileLayer('https://tile.openstreetmap.org/{z}/{x}/{y}.png', {
    maxZoom: 19,
    attribution: '© OpenStreetMap'
}).addTo(map);
// Adiciona marcador no ponto de referência
const marcadorRef = L.marker([pontoRef.lat, pontoRef.lng])
    .addTo(map)
    .bindPopup('<b>São Paulo</b><br>Ponto de Referência')
    .openPopup();
// Função para calcular distância usando Haversine
function calcularDistancia(coord1, coord2) {
    function toRad(x) {
        return x * Math.PI / 180;
    }

    const R = 6371; // Raio da Terra em km
    const dLat = toRad(coord2.lat - coord1.lat);
    const dLon = toRad(coord2.lng - coord1.lng);
    const lat1 = toRad(coord1.lat);
    const lat2 = toRad(coord2.lat);

    const a = Math.sin(dLat/2) * Math.sin(dLat/2) +
              Math.sin(dLon/2) * Math.sin(dLon/2) * Math.cos(lat1) * Math.cos(lat2);
    const c = 2 * Math.atan2(Math.sqrt(a), Math.sqrt(1-a));
    return R * c;
}

// Obtém localização do usuário
if (navigator.geolocation) {
    navigator.geolocation.getCurrentPosition(
        (position) => {
            const userLat = position.coords.latitude;
            const userLng = position.coords.longitude;
            
            // Adiciona marcador da localização do usuário
            const marcadorUsuario = L.marker([userLat, userLng])
                .addTo(map)
                .bindPopup('Sua Localização')
                .openPopup();

            // Calcula e exibe a distância
            const distancia = calcularDistancia(
                {lat: userLat, lng: userLng},
                pontoRef
            );

            // Atualiza a informação de distância
            document.querySelector('.info-distancia').innerHTML = 
                `Você está a aproximadamente ${distancia.toFixed(1)} km do ponto de referência.`;

            // Ajusta o zoom para mostrar ambos os pontos
            const bounds = L.latLngBounds([[userLat, userLng], [pontoRef.lat, pontoRef.lng]]);
            map.fitBounds(bounds, {padding: [50, 50]});

            // Desenha uma linha entre os pontos
            const linha = L.polyline([
                [userLat, userLng],
                [pontoRef.lat, pontoRef.lng]
            ], {
                color: 'blue',
                weight: 2,
                opacity: 0.7
            }).addTo(map);
        },
        (error) => {
            document.querySelector('.info-distancia').innerHTML = 
                'Não foi possível obter sua localização. Por favor, permita o acesso à sua localização.';
        }
    );
} else {
    document.querySelector('.info-distancia').innerHTML = 
        'Seu navegador não suporta geolocalização.';
}
// Adiciona círculo ao clicar no mapa
map.on('click', function(e) {
    // Remove círculo anterior se existir
    if (window.lastCircle) {
        map.removeLayer(window.lastCircle);
    }

    // Cria novo círculo
    window.lastCircle = L.circle(e.latlng, {
        color: 'red',
        fillColor: '#f03',
        fillOpacity: 0.5,
        radius: 500
    }).addTo(map);

    // Mostra popup com coordenadas
    L.popup()
        .setLatLng(e.latlng)
        .setContent(`Latitude: ${e.latlng.lat.toFixed(4)}
Longitude: ${e.latlng.lng.toFixed(4)}`)
        .openOn(map);
});
