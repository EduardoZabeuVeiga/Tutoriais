Menu de Contexto Customizado

Guia prático para criar comandos no Fundo, Arquivos e Pastas
Voltar ao Índice

Visão geral

Este tutorial explica como adicionar itens personalizados ao menu do botão direito no Windows. O método usa o Registro (Regedit) em HKEY_CURRENT_USER\Software\Classes\..., não requer admin e é seguro.

🎯 Onde você quer clicar? (A Regra de Ouro)

O local onde você cria a chave no registro muda dependendo de onde você quer clicar com o mouse.

📂 Em cima de uma PASTA

Ex: Renomear a pasta, Carimbar versão.

Caminho: Directory\shell

Note que NÃO tem "Background".

Argumento: "%V"

O %V envia o caminho da pasta clicada.

📄 Em cima de um ARQUIVO

Ex: Abrir arquivo, Converter imagem.

Caminho: *\shell

A pasta se chama literalmente asterisco (*).

Argumento: "%1"

O %1 envia o caminho do arquivo clicado.

🖱️ No VAZIO (Fundo da Pasta)

Ex: Abrir terminal aqui, Criar estrutura.

Caminho: Directory\Background\shell

Note a pasta extra "Background".

Argumento: "%V"

O %V envia o caminho da pasta onde você está.

Passo a Passo (Como criar)

  1. Escolha o caminho

    Abra o regedit e navegue até HKEY_CURRENT_USER\Software\Classes\. Escolha a subpasta com base na tabela acima (ex: Directory\shell para pastas).

  2. Crie a chave do item (O Nome)

    Clique com botão direito em shell → Novo > Chave. Dê um nome interno (ex: versionamento).

    No lado direito, edite o (Padrão) para o nome que vai aparecer no menu (ex: Carimbar Versão).

  3. Crie a chave do comando

    Dentro da pasta que você acabou de criar, crie outra Chave chamada obrigatoriamente de command.

  4. Defina o Script

    Dentro de command, edite o (Padrão) com o código:

    wscript.exe "G:\Meu Drive\workspace\script.vbs" "%V"
    Atenção: Se for para Arquivos (*), troque o final "%V" por "%1".
Pronto! Não precisa reiniciar, o menu aparece na hora.

Exemplos de Códigos

Script VBS (Pastas)

wscript.exe "C:\Scripts\vers.vbs" "%V"

Script VBS (Arquivos)

wscript.exe "C:\Scripts\vers.vbs" "%1"

Arquivo .BAT

cmd.exe /c ""C:\Script.bat" "%V"""

PowerShell

powershell.exe -File "C:\Scpt.ps1" "%V"

Melhorias úteis (opcionais)

Ícone no menu

Na sua chave do item (ex: versionamento), crie um valor de cadeia (REG_SZ) chamado Icon:

(REG_SZ) Icon = "C:\Caminho\icone.ico"

Mostrar somente com Shift

Crie um valor de cadeia vazio chamado Extended. O item só aparece ao segurar Shift.

Posicionar no topo

Crie um valor de cadeia Position com o texto Top.
