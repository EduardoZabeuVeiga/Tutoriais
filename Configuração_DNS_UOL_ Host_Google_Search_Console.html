Configuração de DNS: UOL Host para Google Search Console

Passo 1: Selecionar o Tipo de Propriedade

Ao acessar o Google Search Console pela primeira vez, você verá uma tela para escolher como deseja verificar seu site.

O que significa cada aba?

Domínio (Esquerda): Esta é a opção mais completa e recomendada. Ela verifica todas as variações do seu site de uma vez (com e sem WWW, HTTP e HTTPS). É a que exige a configuração de DNS (TXT) que estamos fazendo.

Prefixo do URL (Direita): Verifica apenas uma versão exata do site (ex: apenas https://www.meudominio.com). É mais limitada, embora ofereça métodos de verificação mais simples.

O que preencher?

Você deve usar a aba Domínio. No campo de texto, digite apenas o nome do seu site (exatamente meudominio.com) e clique em CONTINUAR.

Passo 2: Copiar o Código TXT do Google

Após inserir o domínio e continuar, o Google abrirá a janela de instruções. Clique no botão COPIAR para obter o código exato.

google-site-verification=SEU_CODIGO_COPIADO_AQUI

Passo 3: Acessar a Zona de DNS do seu Provedor

Faça login no painel onde seu domínio está registrado e acesse a tela de edição das entradas de DNS. Escolha o seu provedor abaixo:

UOL Host, Registro.br, Cloudflare, HostGator, Hostinger, GoDaddy

Nota: As instruções e imagens a seguir mostram o UOL Host, mas os dados a serem preenchidos (TXT e o código) são os mesmos em qualquer plataforma.

Passo 4: Localizar o DNS e Criar a Entrada TXT

Agora usando UOL Host como exemplo, siga este caminho para encontrar a área de DNS e adicionar o código:

  1. No menu de navegação, procure pela secção Domínios ou Meus Domínios.
  2. Selecione o seu domínio (meudominio.com) e clique em Gerenciar DNS ou Alterar Zona de DNS.
  3. Dentro do Gerenciamento de DNS, localize e clique no botão Nova Entrada (ou Adicionar Registro), que costuma estar no topo ou no fim da lista de registos.
  4. Um formulário em branco aparecerá. Preencha os campos estritamente conforme a tabela abaixo:
Campo O que preencher
Selecionar tipo de entrada TXT
Entrada Deixe este campo em branco
Valor (obrigatório) Cole o código completo do Google (ex: google-site-verification=...)
Status da entrada Ativada

Passo 5: Salvar e Aguardar Propagação

Clique no botão Salvar entrada. A nova linha aparecerá na sua lista de DNS.

Aguarde: As alterações de DNS no UOL Host demoram cerca de 1 hora para serem aplicadas na internet.

Passo 6: Concluir a Verificação

Após aguardar o tempo informado, volte à tela do Google Search Console e clique no botão VERIFICAR.
